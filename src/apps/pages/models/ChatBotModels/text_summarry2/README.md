## **Text Summarization using NLP**
The Backend of the project is built in Python using NLP model Transformer and the frontend is built with Streamlit . The Backend and frontend is connected with streamlit in python

### 🎯 **Goal**

Python Project - This  Model is a Natural Language Processing (NLP) model to summarize long pieces of text into concise summaries

Modules Used:
    1. Transformers
    2. Streamlit 

# Text Summarization NLP model Deployment with Streamlit

## Initial Setup:
```
Necessary Modules to install 
1. pip install Transformers
2. pip install tf-keras 
3. pip install streamlit

```
## Run 
```
python -m streamlit run app.py
```
